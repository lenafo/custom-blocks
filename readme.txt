Muestra los productos más vendidos de WooCommerce como bloque Gutenberg.

== Description ==

Este plugin añade un bloque de Gutenberg personalizado que muestra los productos más vendidos de WooCommerce.

== Installation ==

1. Subí la carpeta del plugin a `/wp-content/plugins/`
2. Activá el plugin desde el panel de administración
